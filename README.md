# Tic-Tac-Toe Game

A simple and fun Tic-Tac-Toe game built to practice core programming and game logic concepts. This project is designed with a clean interface and smooth gameplay, making it perfect for beginners as well as for showcasing basic game development skills.

---

## Project Description

This Tic-Tac-Toe Game is a classic two-player game where users take turns marking X and O on a 3×3 grid. The game automatically checks for winning conditions and displays the result when a player wins or when the match ends in a draw. It is designed to be lightweight, fast, and easy to play.

---

## Features

-  Two-player gameplay
-  Real-time win detection
-  Draw match detection
-  Simple and attractive UI
-  Reset and replay option

---

## Technologies Used

- HTML – for game structure  
- CSS – for styling and layout  
- JavaScript – for game logic and interaction  

---

## Learning Outcome

- Understanding game logic
- Working with conditions and loops
- DOM manipulation using JavaScript
- Handling user interactions
